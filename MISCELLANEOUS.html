PREFACE - Alerts

Alerts

1.0 STATIC
Interaction

  • Appears after submit. Alert and errors fade-in on submit.
  • Appears at the top of a form pushing content down
  • Stacks when viewport size is less than max-width

Styles - Container

  • max-width: 600px - applies to 1024px width and larger - min-width: scales to 80% device width
  • height: auto
  • border: 2px solid #D0021B
  • border-radius: 5px
  • background-color: #ffffff

Styles - Text

  • placeholder text - Inherits paragraph pattern
  • wide error - uses paragraph size
  • color:#D0021B

Styles - Headline

  • mobile form factor inherits h2
  • color: #D0021B

Styles - Icon

  • icon scales larger on mobile

5.0 ACTION
Interaction

  • Appears when an action may take place. In this example it is undo
  • Undo appears after an item has been removed from cart or cart item has been saved.
  • Action panel fades in
  • User may dismiss by tapping the X or interacting with anything outside of panel. - Has a five second timer.
  • Is anchored to the bottom of the viewport centered. On mobile viewport it is full width.

Style

  • Inherits design from valid
  • Border: 2px solid #137ABD
  • Inherits affordance link button style

PREFACE - Header & Footer

Header Footer

1.0 HEADER

Interaction

  • header expands full container width
  • logo is left aligned, with option to be center aligned
  • on smaller screens logo is left aligned to allow for room for cart
  • logo links to brand home page
  • In cart - mini cart is not accessible
  • cart icon see pattern library icons
  • background color, border and drop shadow extend 100% width

Styles

  • header container max-width: 1024px - logo and cart cap at edges of this - max-height: 120px;
  • logo max-height: 100px;
  • logo min-padding 10px;
  • background color optional
  • header bottom drop shadow optional
  • header border optional

2.0 FOOTER

Interaction

  • footer expands full container width
  • background color, border and drop shadow extend 100% width
  • uses link style and interaction
  • links wrap on small form and stack on top of each other. If text is too long cashstar and trustwave will stack underneath links
  • cashstar logo drops powered by copy on small device size — logo links to cashstar.com (may want to consider a more descriptive source url opens in new browser
  • use trustwave logo- uses code provided by trustwave

Styles

  • header container max-width: 1024px - logos and text cap at edges
  • background color optional
  • top drop shadow optional
  • border optional
  • logo is svg
