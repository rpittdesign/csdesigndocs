BUTTONS

Buttons, Ghost Buttons, Button Links

PREFACE

Primary Buttons

1.0 STATIC
Interaction

  • Button will scale according to device size. - Text wraps if past max width
  • Two primary buttons cannot be paired
  • Inherits primary color
  • Has border same color as button
  • Height and Font size change according to device size

Styles

  • max-width: 500px - applies to 1024px width and larger - min height: 50px
  • height: 80px
  • border: 1px solid #168DDB
  • border-radius: 6px
  • background-color: #168DDB
  • placeholder text - Sentence case
  • lg button font-size bold 24px, sm button 18px

2.0 HOVER
Interaction

  • On mouse over / hover button background and border color changes to darker shade.
  • Inherits all other button qualities from above.

Styles

  • border: 1px solid #137ABD
  • background-color: #137ABD

3.0 PRESSED
Interaction

  • On press / click border color changes

Styles

  • border: 3px solid #168DDB

4.0 ICON
Interaction

  • Icon aids to tell where a button goes
  • Placement is always on the left in RTL languages

Styles

  • Inherits from 1.0 Static

5.0 AFFORDANCES
Interaction

  • Icon used to indicate it is a button
  • Placement of icon may vary
  • Can include loading icons for loading buttons

Styles

  • Inherits from 1.0 Static
